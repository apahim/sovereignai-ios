What is SovereignAI?

SovereignAI is an iOS client for OpenAI-compatible AI servers. Connect to your own self-hosted LLM (Ollama, vLLM, llama.cpp, and others) or any OpenAI-compatible endpoint and chat from your iPhone or iPad.

Features

  • Connect to any OpenAI-compatible API endpoint
  • Multiple authentication methods (API key, Bearer token, OIDC)
  • Web search, URL fetching, and image search tools
  • Conversation history with export (Markdown, JSON, plain text)
  • Formatted code blocks with copy support
  • Configurable sampling parameters
  • Dark mode support

Privacy first

SovereignAI does not collect analytics, telemetry, or crash reports. Your conversations stay on your device and are only sent to the server you configure. Nothing is sent to us.
